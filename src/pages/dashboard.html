Dashboard de Administración - La Placita Colombiana

Este sistema de administración ha sido desarrollado para La Placita Colombiana, un negocio que comercializa productos provenientes de proyectos de sustitución de cultivos.

Desde este panel, los administradores pueden gestionar el inventario de productos, registrar proveedores, controlar las ventas y visualizar indicadores clave del negocio mediante gráficos en el inicio.
